README
------------------------------
COMPILING THE FOOD MANAGEMNET SYSTEM

To compile either the Food Management System, please do the following: 

1) Open the file directory in the Command Prompt (if not done already).

2) Use the "javac" command followed by the name of the Main.java file.
(ex: >javacMain.java). 
This program does not require any special packages, so compiling the Main 
class will automatically compilie all other classes as well.
------------------------------
EXECUTING THE SYSTEM:

To execute the Food Management System, please do the following:

1) Open the file directory in the Command Prompt (if not done already).

2) Use the "java" command followed by the name of the Main class WITHOUT
the .java or .class file extension.
(ex: >java Main).
